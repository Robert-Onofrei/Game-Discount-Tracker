# Game Discount Tracker

A full-stack application that tracks video game discounts and notifies users when games reach a target price.

## Tech Stack

Frontend:
- React
- TypeScript

Backend:
- FastAPI
- Python

Database:
- MongoDB Atlas

API:
- CheapShark Game Deals API

## Features

- Search games
- Track games in wishlist
- Set target price
- Check for deals
- View store links for purchase

## Architecture

React frontend communicates with a FastAPI backend which stores wishlist data in MongoDB and fetches deal information from the CheapShark API.
